std::size_t  parts,
std::size_t  remainder,
const char *  type 
)

Requests a new piecewise allocation to the server. A number of sequential IDs are allocated according to the part-remainder relations.

If part_size or parts is zero, only the remainder is considered. If remainder is zero, only the evenly sized parts are considered.

Parameters
part_sizesize of each non-remainder part
partsnumber of part_size parts
remaindersize of the last allocation
typeobject type name

first allocation ID, if successful
